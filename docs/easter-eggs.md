## Новые пасхалки

### 1. Konami Code (↑↑↓↓←→←→BA)
- **Сложность:** Low
- **Идея:** Классический Konami Code открывает скрытое меню/режим — например, ускорение анимаций, «турбо-режим» или секретные цвета окон
- **Arch:** добавить еще один keyboard listener EasterEggProvider, по trigger — установка флага `konamiMode` в глобальном состоянии

### 2. Secret Mouse Gestures
- **Сложность:** Medium
- **Идея:** Рисовать буквы/формы мышкой на рабочем столе (как в Opera/Mac OS 9). Например, нарисовать «S» — появляется Skitch-подобное окошко
- **Arch:** потребуется track mouse path, распознавание жестов

### 3. Скрытый about:blank / about:secrets
- **Сложность:** Low
- **Идея:** В Finder window — в адресной строке набрать `about:blank` или `about:secrets` → открывается Easter Egg log

### 4. Пасхалка в 3D Model Viewer
- **Сложность:** Medium
- **Идея:** Нажать определённую комбинацию (R + E + S + E + T?) → модель начинает вращаться/менять цвет/разваливаться
- **Или:** Скрытая 3D-модель (динозаврик из GLTF) появляется при наборе кода

### 5. Дата-зависимые эффекты
- **Сложность:** Low
- **Идея:** 1 апреля — переворот экрана, 25 декабря — снежинки на рабочем столе, 9 сентября (день рождения Mac OS) — специальный салют
- **Arch:** проверка `new Date()` в EasterEggProvider, установка флагов темы

### 6. Сочетание двух пасхалок (Meta-Egg)
- **Сложность:** Medium
- **Идея:** Если найти все 5 пасхалок — открывается секретная 6-я (например, игра Space Invaders / Pong / Breakout прямо в окне)
- **Arch:** useEffect в EasterEggProvider, когда `foundEggIds.length === 5` → показываем секретную иконку на рабочем столе

### 7. Секретный звук / Музыка
- **Сложность:** Low-Medium
- **Идея:** При открытии Time Machine HD — играть короткий MIDI-подобный рифф (System 7 startup sound)
- **Arch:** Web Audio API — генерация простых тонов, без аудиофайлов

### 8. Floppy Disk Easter Egg
- **Сложность:** Low
- **Идея:** Если перетащить файл на иконку «дискеты» (которой нет, но она появляется при определённом клике) — «скопировать» файл на дискету. Потом открыть дискету — там копия

### 9. Разговор с «Desktop AI» (шепотом на системном уровне)
- **Сложность:** High
- **Идея:** По двойному клику на иконке «Happy Mac» (или по sekretnoy комбинации) — появляется окошко с LLM-ассистентом в стиле Mac OS 7
- **Arch:** Потребуется интеграция с API, либо встроенный набор ответов (эмуляция)

### 10. Screen Saver Mode
- **Сложность:** Medium
- **Идея:** Если не двигать мышь 5 минут — запускается скринсейвер в стиле Mac OS 7 (летающие звёздочки/мозаика)
- **Arch:** `useIdleTimer` хук, компонент ScreenSaver с canvas/three.js анимацией

## Условные правила для новых пасхалок

1. **Не перегружать** — каждая пасхалка должна быть маленьким сюрпризом, не основным контентом
2. **Стиль Mac OS 7** — все пасхалки должны выглядеть как часть ретро-системы, не breaking the illusion
3. **Haptics** — каждая пасхалка должна иметь уникальный вибропаттерн
4. **Lazy load** — всё через `React.lazy()` + `Suspense`, никакого влияния на initial bundle
5. **Persist progress** — найденные пасхалки сохраняются в localStorage
6. **Trigger ≠ obvious** — триггеры не должны быть очевидными, но должны быть repeatable (чтобы можно было поделиться)
