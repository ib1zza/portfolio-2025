# Classic Macintosh Portfolio (2025)

An interactive, immersive 3D portfolio that recreates a classic Macintosh System 6/7-style desktop environment. Built with React 19, Vite 7, TypeScript 5.9, Zustand 5, Framer Motion, and React Three Fiber.

## ⚡ Quick Start

For quick reference, standard commands to spin up the local development setup or check files are:

```bash
# Install dependencies
yarn install

# Start development server
yarn dev

# Build for production
yarn build

# Run project linter
yarn lint
```
